# 📊 Strategic Product Placement Analysis

## 🔍 Overview
This project analyzes the impact of product placement, pricing, promotions, and customer behavior on sales performance using interactive data visualization.

The project is built using **Tableau for visualization** and **Flask + Bootstrap for web integration**, making it a complete end-to-end data analytics solution.

---

## 🎯 Objectives
- Analyze the relationship between product placement and sales
- Identify high-performing product categories
- Study customer demographics and purchasing behavior
- Compare product pricing with competitor pricing
- Evaluate the impact of promotions and foot traffic
- Deploy the dashboard using a web application

---

## 🛠 Tech Stack
- Python (Flask)
- Tableau Public
- HTML, CSS, Bootstrap
- Git & GitHub

---

## 📊 Features
- 📈 Interactive Tableau Dashboard
- 📖 Story-based data insights
- 🎛 Filters for dynamic analysis
- 🌐 Web integration using Flask
- 📱 Responsive UI using Bootstrap
